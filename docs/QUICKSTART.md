# Quickstart (60 Seconds)

1. **Launch**: Open the application.
2. **Onboard**: Select "Engineer" role in the Concierge.
3. **Task**: Choose "CodeGen" and "Experimentation".
4. **Generate**: Enter a word like "SYNTH" and choose "Neon Cyber".
5. **Wait**: Video compiles in ~45s.
6. **Deploy**: Copy the self-contained HTML embed code.
