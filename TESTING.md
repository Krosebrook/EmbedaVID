# Testing Matrix

## PWA Validation
1. **Lighthouse:** Run Chrome DevTools Lighthouse -> PWA check.
2. **Offline Mode:** Toggle "Offline" in Network tab; verify app shell loads.
3. **Installability:** Verify "Install App" prompt appears in Chrome/Safari.

## UI/UX
- **Contrast:** Check accessibility contrast ratios in `App.tsx` preview component.
- **Responsiveness:** Test on mobile (iPhone/Android standalone modes).

## Media Generation
- Verify GIF encoding speed and video playback synchronization.
